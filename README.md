# Лабораторная работа №2 — RunTrackerUUST

## Описание

RunTrackerUUST — Android-приложение для отслеживания пробежек. Приложение записывает маршрут через GPS, отображает его на карте и показывает статистику в реальном времени.

## Функции

* запуск и остановка записи маршрута;
* отображение маршрута на карте (OpenStreetMap);
* отслеживание дистанции в километрах;
* отображение текущей скорости;
* отображение высоты над уровнем моря;
* таймер пробежки;
* маркеры старта и финиша на карте.

## Технологии

* Java
* Android SDK
* XML
* LocationManager / GPS
* osmdroid (OpenStreetMap)
* Polyline / Marker

## Разрешения

```xml
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.INTERNET" />
```

## Инструкция по запуску

1. Клонировать репозиторий:
```bash
git clone https://github.com/Ja1k3r37/RunTrackerUUST.git
```
2. Открыть проект в Android Studio.
3. Подключить реальное устройство (для точного GPS-сигнала).
4. Нажать **Run**.

## Использование

1. Запустить приложение.
2. Нажать кнопку **Начать** — приложение запросит разрешение на геолокацию.
3. Выйти на пробежку — маршрут будет отображаться на карте, статистика обновляется каждые 2 секунды.
4. Нажать **Стоп** — на карте появятся маркеры старта и финиша.
